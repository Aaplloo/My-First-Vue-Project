# 规划计划与投资管控平台 - 演示说明

## 🎯 项目概述

本项目是基于Vue3开发的规划计划与投资管控平台，完整实现了Axure设计稿的功能和外观，已优化为响应式布局，最佳显示效果为1920*1080分辨率。

## 🚀 快速开始

### 1. 启动项目
```bash
npm run dev
```

### 2. 访问地址
打开浏览器访问: `http://localhost:5174`

### 3. 最佳显示效果
- **推荐分辨率**: 1920*1080
- **浏览器**: Chrome、Firefox、Edge
- **显示模式**: 全屏或最大化窗口
